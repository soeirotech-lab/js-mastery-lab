# JS Mastery Lab 🚀

> 🇺🇸 A public JavaScript engineering laboratory for technical mastery, problem solving, clean code, debugging, and career preparation.
>
> 🇧🇷 Um laboratório público de engenharia em JavaScript focado em maestria técnica, resolução de problemas, código limpo, debugging e preparação profissional.

---

# 🇺🇸 English Version

## 📌 About the Project

**JS Mastery Lab** is a long-term technical training repository created to improve JavaScript fluency through:

- daily exercises
- debugging practice
- clean code studies
- interview preparation
- architecture thinking
- algorithm drills
- reusable utilities
- AI-assisted learning

This repository is part of the **SoeiroTech Lab Ecosystem**.

The main goal is not only to learn JavaScript, but to build professional engineering habits and a public learning history.

---

## 🎯 Main Goals

- Improve JavaScript fluency
- Practice problem solving
- Learn professional Git/GitHub workflow
- Build technical discipline
- Create a strong portfolio
- Study software engineering concepts
- Prepare for technical interviews
- Document long-term learning

# 📁 Project Structure

| Folder                  | Description                                                  |
| ----------------------- | ------------------------------------------------------------ |
| `time-trials/`          | Timed challenges focused on speed and logical reasoning.     |
| `mock-exams/`           | Simulated tests, coding assessments and technical exercises. |
| `snippets-and-tricks/`  | Reusable snippets, syntax tricks and helper solutions.       |
| `clean-code-patterns/`  | Clean Code, SOLID, Design Patterns and architecture studies. |
| `sandbox-debugging/`    | Concept laboratory for debugging and JavaScript internals.   |
| `routine-drills/`       | Daily repetition exercises for JavaScript fundamentals.      |
| `interview-prep/`       | Technical interview preparation and live coding practice.    |
| `youtube-insights/`     | Exercises and notes inspired by dev tutorials and videos.    |
| `ai-assisted-learning/` | Experiments and exercises assisted by AI tools.              |
| `utils-and-helpers/`    | Personal utility functions and reusable helpers.             |
| `docs-and-wiki/`        | Notes, documentation, summaries and study materials.         |
